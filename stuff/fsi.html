Olá!

Este é um enigma com 10 fases (ou 11, se contar a última de bônus). Cada fase tem uma palavra aléatória como resposta, e você terá que descobrir ela por conta própria.


A cada fase que você passa, o enigma fica mais difícil. As fases de 1-5 são as mais fáceis. As fases de 6-8 são mais difíceis. As fases 9-11 são as mais difíceis de todo o enigma.


* Para as respostas, não use hífen nem acento. Ex.: guarda-roupa -> guarda roupa; amanhã -> amanha.

* Números devem ser escritos por extenso. Ex.: 5 -> cinco; 99 -> noventa e nove.
